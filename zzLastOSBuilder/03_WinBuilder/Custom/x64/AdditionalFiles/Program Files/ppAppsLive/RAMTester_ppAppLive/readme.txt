features:
- two testing methods (random and shuffle addressing)
- i18n support. Russian and Italian (thanks to Marco Sprocatti) languages are available.
- x86/x64 platforms native support
- MT/SMP ready
- Win9x/WinNT+ compatible (tested under Win XP SP2 and Win XP RC 64bit edition only)
- emmbedded memory compacting function


configuration:
language selection support is stored in the Registry and other tweaks are given 
from rt.ini file.
themes  - XP themes support
timeout - working threads sleep time
compact - memory compacting is available from system menu
confirm - some extra yes/no added


files:
rt.exe - RAMTester utility executable
cpl.txt - license
readme.txt - this file
x64/ - version for x64 OS
i18n/ - alternate languages support files for both x86/x64
(copy into application directory and register with .reg file)
